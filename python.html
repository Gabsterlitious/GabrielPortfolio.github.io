Lab Runner

Description: Use Python to create a moving guy on the screen. He needed to respond to all 4 arrow keys and move in the corresponding direction. In addition, we also had to limit the guy to just the screen.

Concepts Learned:
Drawing- Getting the Guy to Show up on the Screen
Class- Create the person class with the correct image
While loop- While true, continue
Arrow keys- how to have the program recognize the keys

Wall Runner

Description: We had to create two guys on the screen along with a wall. One guy had to be controled with the arrow keys and the other had to be controled with WASD. We needed to create boundaries, along with having the guys run into eachother and the wall.

Concepts Learned:
Draw- We learned how to draw more than one person
Changed Recs- We learned hoe to get the position of the other objects
Collide- We learned how to make the objects recognize they are hitting eachother

Ground Runner

Description: For this lab, we had to create a game with a timer. We made a guy run-using the arrow keys-after randomly appearing coins to gain points. Once you run out of time, Game ended would appear, and if the player got 10 coins before the time ran out, it would display You Win!!!

Concepts Learned:
Timer- How to use and display the timer
Random Variant- Make the coins pop up in random positions
Background- Change the background
Score- Variable to keep track of and display the score

Dodge Runner

Description: For this game, we had to make a game where you had to cross the road with your guy. You just control him with the arrow keys, but the tricky part was that there were aliens crossing. The game got a lot harder every level and it eventually became extremely difficult.

Concepts Learned:
Speed- How to change the speed of the aliens
Random Variant- Make the aliens come at random places
Game Win and Game End

Final Project

Description: For this game, we had to make our own game using the resources we had. I made a game that when you click the spacebar, the soccerball would jump up, and it would fall otherwise. You had to try to get the ball into the goal and it had 3 different levels as you went.

Concepts Learned:
Background- How to make a moving background
Levels- How to change levels
Title Page- how to make a title page
Movement- How to make the ball keep going down unless you hit the spacebar
Sound- How to get music in the game
